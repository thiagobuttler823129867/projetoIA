SOBRE

Usuário: Profissional de Marketing Digital.

Interface: Interface Gráfica (GUI);

Lógica: Com base nas avaliações/venda é realizada um comparativo entre ambos e a partir disso, é gerado uma métrica de quais produtos vendem mais/menos e são melhores/menos avaliados. Ex: O produto X vende muito e está bem avaliado

Dados: Quantidade de vendas, revendas, avaliações, clientes, etc.

Requisitos Funcionais: - Apresentar dados de vendas dos produtos; - Sugerir produtos tendência no mercado; - Analisar estratégias de venda e sugerir melhorias;

Requisitos Não Funcionais: - Dados apresentados em formato de site (Html, JavaScript, CSS); - Interface intuitiva; - Permitir portabilidade para todos dispositivos/browsers;”
